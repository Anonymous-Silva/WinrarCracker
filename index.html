How to Use the Script

  1. Install WinRAR: Ensure WinRAR is installed on your system (unrar.exe must be in "C:\Program Files\WinRAR\"). Download it from the official site if needed.
  2. Download the Script: Click the button above to download crack_rar.bat.
  3. Place the RAR File: Have your password-protected RAR file ready on your computer.
  4. Run the Script: Double-click crack_rar.bat or run it via Command Prompt. Drag and drop your RAR file into the terminal or paste its full path when prompted.
  5. Wait for Results: The script will try numeric passwords from 0 to 9999. If the password is found, it will display it; otherwise, it will stop after 9999 attempts.
  6. Check Output: If successful, the RAR contents will be extracted to a temporary folder, and the password will be shown. The temp folder is deleted automatically.

Notes

  • This script only tries numeric passwords (0-9999). Modify it for other ranges or character sets if needed.
  • Ensure the RAR file path has no spaces or use quotes if it does.
